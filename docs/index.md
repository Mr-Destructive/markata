## QuickStart

### Create Some Content

```
mkdir pages
echo '# My First Post' > first-post.md
echo '# Hello World' > hello-world.md
```

### Build your site

``` bash
pip install markata
markata build

# or if pipx is your thing
pix run markata build
```
